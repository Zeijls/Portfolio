Progressive Web Apps

Minor Web Development

Voor dit project moest de web aplicatie moet server side worden gerenderd. Zodra de gebruiker offline is, moet de content zichtbaar blijven. De content die de gebruiker al een keer eerder heeft bezocht kan offline beschikbaar zijn doordat het wordt opgeslagen in de cache. Het Critical Rendering path wordt geoptimaliseert. Na het verwerken van bovenstaande criteria moet het een progressive web app zijn. Ik heb deze criteria toegepast op een ouder project van een ander vak, en de website geoptimaliseert.
