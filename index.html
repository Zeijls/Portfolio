Simone van Zeijl

Student Communication and Multimedia Design

Hi! Welkom om mijn website. Ik ben Simone van Zeijl en zit in het derde jaar van de opleiding Communcatie en Multimedia Design - aan de Hogeschool van Amsterdam.

Mijn focus ligt op frontend development. Ik volg op dit moment de Minor Web Development aan de HvA. De combinatie tussen frontend en design trekt mij ontzettend aan.

Naast mijn studie heb ik mijn eigen bedrijf opgericht: SiZeD. Hier is werk van te zien in mijn portfolio. Ik ben beschikbaar als freelance frontend developer.

Ik vind het ontzettend belangrijk om mezelf te blijven ontwikkelen, en op deze manier iedere dag een stukje beter te worden.

Portfolio

Projects I Worked on

Real Time Web

Minor Web Development

De opdracht was om customized real time events te ontwikkelen door middel van Socket IO. De data is afkomstig uit de Spotify API. In mijn applicatie is het mogelijk om de Spotify Game te spelen. Voor dit spel is een premium account op Spotify vereist. In het spel kun je samen met je vrienden nummers raden en punten verdienen.

Browser Technologies

Minor Web Development

Voor dit project kreeg ik de opdracht een enquete te ontwikkelen die ten alle tijden kan worden opgeslagen. Deze enquete moest door middel van progressive enhancement worden opgebouwd. De eerste laag was functional, de website moest voor iedere gebruiker toegankelijk zijn dus zonder javascript en zonder styling. Toch moest de enquete kunnen worden opgeslagen, hier heb ik een oplossing voor bedacht. In de tweede laag wordt de CSS toegevoegd en in de derde laag de Javascript.

Web Design

Minor Web Development

Om de groene energie in huis bij te houden is de volgende applicatie ontwikkelt voor Roger Ravelli. Er is audioondersteuning aanwezig zodat hij met zijn oogaandoening de progressie van zijn zonnepanelen kan bijhouden zonder hier erg vermoeid van te raken.

Progressive Web Apps

Project Information Design

Voor dit project moest de web aplicatie server side worden gerenderd. Zodra de gebruiker offline is, moet de content zichtbaar blijven. De content die de gebruiker al een keer eerder heeft bezocht kan offline beschikbaar zijn doordat het wordt opgeslagen in de cache. Het Critical Rendering path wordt geoptimaliseert. Na het verwerken van bovenstaande criteria moet het een progressive web app zijn.

CSS To The Rescue

Minor Web Development

Deze website komt een beetje vreemd over zonder context. Voor deze opdracht kon je een aantal richtlijnen kiezen, verder ging het vooral om het experimenteren met CSS. Voor dit vak ben ik de Almanac van CSS tricks afgegaan, en alles wat me interessant leek ben ik mee gaan uitproberen. Ik heb ontzettend veel nieuwe tricks en elementen van CSS geleerd.

Controle Alt Delete

Project Information Design

Controle Alt Delete was de opdrachtgever voor ons eindproject van Information Design. Het bedrijf heeft onderzoek gedaan naar het Etnisch Profileren van de politie. Dit onderzoek is uitgevoerd door middel van een enquete in Amsterdam. Voor ons was de opdracht om de uitkomst van dit onderzoek visueel weer te geven in datavisualisaties. Controle Alt Delete wilde naderhand het project samen met ons team uitwerken. Zie de uitwerking onder de monitor.

Infographics

Project Information Design

We kregen een routebschrijving van de weg die de gebruiker moest afleggen op de Campus, hierin waren een aantal handelingen verwerkt. Aan ons de opdracht om dit visueel weer te geven in een infographic. Tegelijkertid hadden we ook het vak motion design waarin we dezelfde routebeschijving moesten animeren. Vandaar dat u het filmpje van Motion Design kunt bekijken als u de QR code scant. De poster is gemaakt in Adobe Illustrator.

Motion Design

Project Information Design

Voor dit vak hadden we dezelfde routebeschrijving ontvangen als bij Infographics. Nu was de opdracht om een klein stukje uit de routebeschrijving met de inbegrepen handelingen visueel weer te geven in een animatie. De animatie is gemaakt in Adobe After Effects. Ik had hier voorheen nog nooit meer gewerkt, en vond het heel leuk om hier mee aan de slag te gaan.

De monitor

SiZeD

Na het project van Information Design op de HvA, had Controle Alt Delete ons team benaderd om het project samen uit te werken zodat zij het konden publiceren. Jair, de opdracht gever had veel punten die hij wilde aanpassen, dit hebben wij toegepast. Controle Alt Delete is voor hun zelf nog een paar kleine puntjes op de i aan het zetten dus de website is nog niet gepubliceerd. Zodra de website online staat zal hij hieronder tevinden zijn.

Xanthu

SiZeD

Uiteindelijk hebben ze hier een periode niet meer naar om gekeken, en moest alle content er nog in geplaatst worden. Ik heb deze content voor hun geplaatst, afbeeldingen bewerkt en veel contact gehad met de bouwer van de website.

Pubquiz Velo

SiZeD

Voor Sport Vereniging VELO in Wateringen kreeg ik de opdracht een poster te ontwerpen voor hun Pubquiz die de damesselectie van de handbal had georganiseerd. De poster is gemaakt in een combinatie van Adobe Indesign en Adobe Illustrator.

Ring the Bottle

Project 2 team

Voor dit project moesten wij een oplossing zoeken voor het Petfles probleem op de AmstelCampus. Er werd teveel onnodig plastic gebruikt doordat de studenten iedere dag een nieuw flesje aanschaffen. Hier heb ik samen met mijn projectgroepje de oplossing voor bedacht. Het eindproduct is een herrineringssysteem om de student er iedere dag aan te helpen herrineren zijn recyclebare fles mee te nemen. Zie video voor het concept.
